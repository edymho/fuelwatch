# ⛽ FuelWatch — Localização e Gestão de Crise de Combustível

> Aplicação web progressiva (PWA) que monitoriza em tempo real a disponibilidade de combustível nas bombas de Moçambique.

## 🚀 Instalação

```bash
git clone https://github.com/edymho/fuelwatch.git
cd fuelwatch
npm install
npm run dev
```

App disponível em **http://localhost:5173**

## Stack
- React 18 + Vite 5
- Recharts, Lucide React  
- Anthropic Claude API (notícias IA)

## Funcionalidades
- 🗺️ Mapa interactivo de bombas em tempo real
- ⛽ Listagem filtrável por combustível e estado
- 💰 Preços oficiais ARENE/MIREME
- 📰 Feed de notícias gerado por IA
- 💬 Chat comunitário com votos
- 📊 Dashboard para proprietários/funcionários
- 🔔 Alertas de crise automáticos

## Cidades
Maputo · Matola · Beira · Nampula · Pemba · Quelimane · Tete · Inhambane
